## What is TVFlix?

TVFlix is a three-page streaming platform frontend. It covers the main screens a streaming site needs — a home page with featured content, a movie list for browsing, and a detail page for individual titles. It includes a search bar with an animated open/close toggle and a responsive mobile navigation menu.

---

## Features

- **Home page** — hero section with featured content and category rows
- **Movie list page** (`movie-list.html`) — full browsable catalog
- **Detail page** (`detail.html`) — individual movie info, poster, and description
- **Search toggle** — search bar that opens and closes with animation
- **Mobile menu** — collapsible navigation for small screens
- **Responsive layout** — works on all screen sizes

---

## How to Use

```bash
start index.html      # Windows
open index.html       # macOS
```

Page flow: `index.html` → `movie-list.html` → `detail.html`
